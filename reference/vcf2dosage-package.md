# vcf2dosage: Polyploid-Aware VCF to Dosage Matrix Conversion for Genomic Selection

Converts Variant Call Format (VCF) files to numeric allele dosage
matrices ready for genomic selection and genome-wide association
studies. Handles diploid and polyploid species (tetraploid, hexaploid,
and arbitrary ploidy), filters variants by minor allele frequency,
missingness, and read depth, and exports dosage matrices in formats
compatible with 'rrBLUP', 'GWASpoly', 'BGLR', 'sommer', and 'AGHmatrix'.
Supports both unphased (0/0/1/1) and phased (0\|0\|1\|1) genotype calls.
